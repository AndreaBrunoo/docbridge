# Dockbridge Premium Demo

Demo frontend autonoma in HTML/CSS/JavaScript.

## Avvio

Apri `index.html` in un browser moderno.

## Contenuti

- Dashboard wow con workflow animato
- Tre aree principali: Consultazione, Area Staging, Caricamento Manuale
- Tabelle paginate e ordinabili
- CSV lato frontend: `esempio-unoenergy.csv`
- Confidence Engine con badge verde/arancione/rosso
- Vincoli hard: commodity, POD/PDR, data firma
- Match automatico e manuale assistito
- Viewer PDF mock e metadata drawer
- Dark mode
- LocalStorage per persistenza demo

## Reset dati

Cancella il localStorage del browser per ricominciare da zero.
